US Traffic Accidents Analysis (2016–2023)
Overview

This project analyzes U.S. traffic accident data (2016–2023) to identify key patterns in accident frequency and severity across time, weather, and road characteristics. The goal is to better understand major risk factors and support data-driven safety insights.

Objectives

Analyze how accident severity varies by time of day and weather conditions

Explore accident frequency trends by state and road type

Identify which factors appear most associated with severe accidents

Dataset

Source: U.S. Traffic Accidents Dataset (2016–2023)

To improve performance and allow easy use in Tableau, the dataset was reduced to key features and aggregated into a lighter file.
